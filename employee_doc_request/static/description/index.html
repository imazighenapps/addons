All features of Module

  1. Allow your employees to request their documents (work certificate, payslip, leave permit, etc.)
  2. Approval or denial of Department Head with reason of rejection.
  3. Allow the manager to create an infinite number of document types
  4. You can specify the document type for specific or all departments
  5. Offer your employee two types to have their documents(as attachment or Hand to hand)
  6. The employee will just see these requests
  7. The head of department will see just the request address to him
