Odoo Module: Dynamic Dashboards

This Odoo module enables you to create dynamic dashboards for your application.

Main Features:

  • Create customized dashboards tailored to your needs.
  • Customize charts, tiles, and key performance indicators (KPIs).
  • Integrate your dashboards into the Odoo interface for easy access.
